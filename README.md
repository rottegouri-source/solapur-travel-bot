# Solapur Travel Guide Bot

AI-powered tourism chatbot for Solapur, Maharashtra.

## Features

* Tourist places guidance
* Hotels & stay suggestions
* Food recommendations
* Weather support
* Transport help
* Itinerary planning

## Technologies Used

* Google Dialogflow
* HTML
* CSS
* GitHub Pages
